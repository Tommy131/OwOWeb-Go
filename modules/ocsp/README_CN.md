# OCSP-Service

[English Version](README.md)

该应用程序基于Go语言编写, 并且需要 [OwOWeb-Go](https://github.com/Tommy131/OwOWeb-Go) 框架的支持.

本应用程序目前包含如下功能:

- [x] 解析CA证书 & 其加密私钥
- [x] 接受 & 处理 OCSP 请求
- [x] 查询证书吊销状态

## 使用方法

1. 在任意目录下使用指令 `git clone https://github.com/Tommy131/OwOWeb-Go .`
2. 使用指令 `go run main.go` 即可启动 `OCSP-Service`.
